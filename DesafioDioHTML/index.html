Tags mais comuns

<p>

 O parágrafo representado ao lado é uma tag muito utilizada.

<mark>

 Essa tag cria uma marcação no texto.

<small>

 Essa tag deixa o texto bem pequeno.

<i>

 Essa tag deixa o texto em itálico.

<u>

Essa tag deixa o texto sublinhado.

<strong>

Essa tag deixa o texto em negrito.

<sub>

Essa tag deixa o texto na parte inferior.

<sup>

Essa tag deixa o texto na parte superior.
